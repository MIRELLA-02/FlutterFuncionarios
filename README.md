# Atividade Funcionários
------------
# ⚙️ Tecnologias Utilizadas
- 📱 Flutter
- 💙 Dart
-----------------
# 🗂️ Sobre o Projeto
Aplicativo mobile desenvolvido com Flutter para gerenciamento de funcionários, permitindo visualizar informações como nome, cargo e imagem.
Projeto criado como atividade prática durante o curso técnico em Desenvolvimento de Sistemas no SENAI.


----------------
# 🖼️ Telas
- 📋 Lista de funcionários
- 🔍 Detalhes do funcionário
- 🎨 Interface simples e intuitiva

---------------
# 🛠️ Como rodar seu código
- Passo 1:  Clone o repositório
git clone "link do seu repositório"

- Passo 2: Acesse a pasta
  cd seu-repositorio

- Passo 3: Instale as dependências
flutter pub get

- Passo 4: Execute o projeto
flutter run
